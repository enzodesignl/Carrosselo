# PostManager — pronto para Vercel

Este projeto contém apenas um arquivo:

- index.html

Como publicar na Vercel:
1. Crie um repositório no GitHub.
2. Envie o arquivo index.html na raiz do repositório.
3. Importe o projeto na Vercel.
4. Clique em Deploy.

Esta versão é leve e não usa imagens pesadas em base64, evitando tela branca no deploy.

A integração real com IA deve ser feita depois por backend/Vercel Functions, nunca colocando a chave da API dentro do HTML.
